## Overview

**QUICKCART** is a complete e-commerce storefront for a grocery and pansar store, built in pure vanilla JavaScript — no frameworks, no build step, just clean, fast, hand-crafted code. It pairs a polished customer-facing website with a full admin panel for managing products, categories, and site content.

It proves that a fast, beautiful, fully-functional store doesn't need a heavy framework — just solid fundamentals.

## Key Features

### Customer Website
- 🛒 **Working shopping cart** — add, remove, update quantities
- 🔍 **Live product search** — instant filtering as you type
- 🎯 **3D tilt effects** on product and feature cards
- 🎭 **Category filter tabs** — browse grocery vs. pansar items
- 🌊 **Smooth scroll animations** — fade-up reveals, animated counters, floating cards
- ⏳ **Animated preloader** with progress ring
- 📱 **Fully responsive** — mobile, tablet, desktop
- 📞 **Contact & newsletter forms**

### Admin Panel
- 🔐 **Login authentication**
- 📊 **Dashboard** with live statistics
- 📦 **Product management** — view, search, filter, edit, delete
- ➕ **Add products** with full details
- 🏷️ **Category management**
- 🌐 **Site info editor** — name, tagline, currency, social links
- 📞 **Contact info editor**
- 💾 **Auto-save** — all changes persist locally

## Tech Stack

| Component | Technology |
|-----------|-----------|
| **Core** | Vanilla JavaScript (ES6+) |
| **Styling** | CSS3 (gradients, animations, 3D transforms) |
| **Data** | Structured `localStorage` store (`data.js`) |
| **Effects** | Custom 3D tilt engine (`tilt.js`) |
| **Build** | None — zero dependencies, instant load |
